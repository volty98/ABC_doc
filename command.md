# Airio-Base Console ABC コマンドリファレンス

## ＊コマンド書式
**command** [-arg1[:param1]] [-arg2[:param2]] [-arg3[:param3]]

### 例1：ledポート1に0(Low)を出力
```
led -port:1 -out:0
```
### 例2：Airio-Baseをソフトリセット
```
reset
```
